Cnversor de Unidades em Go

Aplicação em linha de comando desenvolvida em Go para conversão entre diferentes unidades de medida.

---

## Funcionalidades

O programa permite converter valores entre:

### Temperatura
- Celsius
- Fahrenheit
- Kelvin

### Comprimento
- Metro
- Centímetro
- Quilômetro

### Velocidade
- km/h
- m/s

### Tempo
- Segundos
- Minutos
- Horas

### Volume
- Litros
- Mililitros
- Metros cúbicos

---

## Como executar

1. Tenha o Go instalado
2. Clone o repositório
3. Execute no terminal:

```bash
go run conversor.go
